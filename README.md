# PRODIGY_DS_01
Task1-Data Visualization using Population Dataset 

### Objective
Create visualizations to represent the distribution of a continuous variable using appropriate charts. This project focuses on understanding population distribution across countries through graphical analysis.

### Dataset
Population by Country 2020 Dataset

### Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

### Project Workflow
- Imported and explored the dataset
- Examined the dataset structure and summary statistics
- Created a histogram to visualize the distribution of population
- Generated a bar chart to compare population values across selected countries
- Customized charts with titles, labels, and formatting for better readability

### Files Included
- Task1.ipynb
- population_by_country_2020-selected-columns.csv

### Outcome
Successfully visualized the population distribution using different charts, enabling better understanding of population trends and comparisons across countries.
